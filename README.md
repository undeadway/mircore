# Chinese

## 概要
这是一个基于 Noodejs 的玩具服务器。  
这个服务器只用于个人学习使用。  
demo 文件夹下是服务器的一个简单案例。

## 实现功能
* 可以进行路由控制；
* 拒绝指定客户端访问；

## 启动
```
var server = require("proton");
server.start();
```

## 实际应用
* [路上的垃圾回收](http://waygc.net)
* [道亦会](http://daoyiclub.com)
